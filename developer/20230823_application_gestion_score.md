# Gestion de scores

## Idées

* Gestion score 2 équipes
* Notification sonore sur appui de certains boutons
  * Fin de match
  * mi-temps
  * But
  * Faute
* Ajout du temps
* Doit s'executer sur un appareil Android
* Ne doit pas utiliser de connexion réseau externe
